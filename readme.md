### 🚀 My Workflow

I believe in **GitOps**. Instead of manual configuration, I strive for this workflow in my Homelab and projects:

1. **Code:** Write infrastructure configurations, scripts, or modding logic.
2. **Test:** Validate changes locally or through CI pipelines.
3. **Push:** Commit changes to GitHub.
4. **Deploy:** Apply changes to servers, VMs, or modding environments automatically.
